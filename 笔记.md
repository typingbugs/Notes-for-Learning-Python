## 语法
- 注释
    - 单行注释 **#** 或 **用一对单引号或双引号括起来**
    - 多行注释 **用一对三单引号或三双引号括起来**
        - 本身多行注释就是一个字符串，但是这个字符串并没有在程序中被使用，所以可以当作注释使用
        - 如果字符串中出现单引号或双引号，最外层的引号就要用与之不同的引号括起来，如果都出现，就要用三单引号括起来，或使用转义字符
- 语句
    - 依靠缩进体现语句之间逻辑关系
    - 如果一行语句太长，可以在行尾加续行符'\'来换行分成多行，但是更建议使用括号来包含多行内容
- 变量
    - Python采用基于值的内存管理方式，创建变量不需要事先声明变量名及其类型，直接赋值
    - python有自动内存管理功能，不需要太多考虑内存管理问题，但使用完毕后可以主动用**del(变量)**删除变量
    - 变量名可以是大小写字母、数字、下划线和汉字等字符的组合，命名识别大小写，首字符不能是数字，不能与关键/保留字相同
    - 如"人生苦短我学Python"可以作为一个变量名
    - 变量用"="赋值
        - 如TempStr = "82F"即向TempStr变量赋值"82F"的字符串
    - Python允许多个变量指向同一个值，当其中一个变量修改值后，其内存地址才会发生变化
        - 如x=3,id(x)的值为1786684560，此时赋值y=x,id(y)的值也为1786684560，之后进行x+=6操作，id(x)=1786684752,id(y)=1786684560不变
- 列表(list)类型
    - 用[]表示，元素之间用","分离，表示多个元素，使用元素序号作为下标
        - 如['F','f']表示含有两个字符串的列表，表示字符串'F'或者字符串'f'
    - 元素查找速度非常慢；增删元素速度在尾部操作快，其他位置慢
    - 可以使用“+”运算符将元素添加到列表中
        - 如aList = [ 3, 4, 5 ], aList = aList + [ 7 ]，此时aList的值为[ 3, 4, 5, 7 ]
        - 此操作速度较慢，涉及大量元素添加时不建议使用此方法。因为实际上这不是为列表添加元素，而是创建了一个新列表，并将原列表中的元素和新元素依次复制到新列表的内存空间，涉及大量元素的复制，因此建议用方法append()或extend()实现在尾部添加元素的功能
    - 可以使用“*”运算符将列表重复
        - 此操作并不是复制子列表值，而是复制已有元素的引用，因此修改其中一个值时，相应的引用也会被修改
        - 如x = [ [ [ None ] * 2 ] * 3 ]，x的值为[ [ None, None ], [ None, None ], [ None, None ] ]，x[0][0] = 5，x的值为[ [ 5, None ], [ 5, None ], [ 5, None ] ]
    - 可以使用切片来原地修改列表内容
        - 如aList = [ 3, 5, 7 ], aList[len(aList):] = [ 9 ]实现在尾部追加元素，此时aList = [ 3, 5, 7, 9 ]，aList[:3] = [ 1, 2, 3 ]替换前3个元素，此时aList = [ 1, 2, 3, 9 ]，aList[::2] = [ 0 ] * 2替换偶数位置上的元素，此时aList = [ 0, 2, 0, 9 ]，aList[:3] = []删除前3个元素，此时aList = [ 9 ]
    - 切片操作是列表对象的浅复制，即生成一个新的列表，并且把原列表中所有的元素的引用都复制到新列表中。
        - 非浅复制：如aList = [ 3, 5, 7 ], bList = aList *(非浅复制)*，此时bList和aList指向同一个内存，修改其中一个会影响另外一个，如bList[1] = 8，则aList的值为[ 3, 8, 7 ]，aList == bList的值为True，aList is bList的值为True，是同一个对象
        - 浅复制：如aList = [ 3, 5, 7 ], bList = aList[::] *(切片，浅复制)*，aList == bList的值为True，表明两个列表元素完全一样，但aList is bList的值为False，不是同一个对象，内存地址不一样，修改其中一个不会影响另一个
        - 但是如果原列表中包含列表之类的可变数据类型，由于浅复制时只是把子列表的引用复制到新列表中，这样的话修改任何一个的子列表都会影响另一个的子列表
            - 如x = [ 1, 2, [ 3, 4 ]], y = x[:], x[0] = 5，x的值为[ 5, 2, [ 3, 4 ]]，y的值为[ 1, 2, [ 3, 4 ]]，此时进行x[2].append(6)操作，则x的值为[ 5, 2, [ 3, 4, 6 ]]，y的值为[ 1, 2, [ 3, 4, 6 ]]
        - 列表推导式 使用简洁的方式快速生成满足特定需求的列表，代码具有非常强的可读性
            - 语法形式为：**[ expression for expr1 in sequence1(\n) if condition1 for expr2 in sequence2 if condition2(\n) for expr3 in sequence3 if condition3 ]**
            - 如aList = [ x\*x for x in range(10) ]相当于aList = [] for x in range(10): aList.append(x\*x)，也相当于aList = list(map(lambda x: x\*x, range(10)))
- 元组(tuple)类型
    - 用圆括号()表示，元素之间用“,”分隔，使用元素序号作为下标。用“=”将一个元组赋值给变量，如果只包含一个元素，必须在元素的后面加逗号
        - 如a_tuple = ('a', 'b', 'mpilgrim', 'z', 'example')，a_tuple = (3,)
    - 一旦创建，用任何方法都不能改变其元素，也不允许增删元素
        - 从效果上看，tuple()冻结列表（对不需要修改的元素冻结，使代码更安全），list()融化数组
        - 元组不可变，因此可以作为字典的键或集合的元素；而列表可变，因此不能作为字典的键或集合的元素
    - 查找速度很慢，但比列表快
        - 如果定义了一系列常量值，程序仅对其进行遍历操作时，一般使用元组存储
    - 对元组序列解包可以对多个变量同时赋值
        - 如v_tuple = (1, 2, 3), (x, y, z) = v_tuple
        - 可以进行序列解包的对象不止元组
            - 常数对象，如x, y, z = 1, 2, 3
            - range对象，如x, y, z = range(3)
            - 迭代器对象，如x, y, z = iter([1, 2, 3])
            - 可迭代的map对象，如x, y, z = map(str, range(3))
            - 变量对象（可以交换值），如a, b = b, a
            - sorted()函数返回的有序列表对象，如x, y, z = sorted([1, 3, 2])
            - 字符串对象，如a, b, c = 'ABC'
            - 字典对象，如s = {'a':1, 'b':2, 'c':3}, b, c, d = s.items()，得到b = ('a', 3)
            - 对列表切片修改也支持序列解包，如x = [1, 2, 3, 4, 5, 6], x[:3] = map(str, range(5))，得到x = ['0', '1', '2', '3', '4', 4, 5, 6]
            
- 字典(dic)类型
    - 用大括号{}表示，是可变类型，元素之间用“,”分隔，每个元素(item)由键和值(value)组成，使用键作为下标，且键必须可哈希(不可变)；元素查找、增删速度快
- 集合(set)类型
    - 用大括号{}表示，是可变类型，元素必须可哈希，元素之间用逗号分隔，但不支持下标；查找速度非常快，增删元素速度快
- 字符串
    - 用单引号或双引号括起来的内容表示一个字符串
    - Python中字符串是不可变类型
    - 索引：在字符串后面加[ ]
    - 正向递增序号：从0开始往右数;反向递减序号：从最右端开始，倒数第一个字符序号是-1，依次是-2，-3，-4，反向递减
    - **<字符串>[M: N: K]** 字符串切片，取出序号为M到N(M闭N开)的字符，K表示步长，即取字符周期
        - 如对字符串："〇一二三四五六七八九十"切片[1: 8: 2]结果是"一三五七"
        - M缺失表示从开头开始，N缺失表示到结尾结束，K可以取负数
        - 特殊用法：对字符串："〇一二三四五六七八九十"切片[: : -1]的结果是"十九八七六五四三二一〇"
        - 字符串操作符
            - **x + y** 连接两个字符串x和y
            - **n * x**或**x * n** 复制n次字符串x
            - **x in s** 判断x是否是s的子串语句，若真返回True，假返回False
    - 字符串前面加r或R表示原始字符串，即特殊字符'\'不转义，但是最后一位str[-1]不能是'\'
        - 主要用于正则表达式、文件路径或者URL的场合
- 转义字符
    - **\\"表示字符"**，**\b**回退，**\n**换行，**\r**回车（光标移到本行首）
- 4种常用进制
    - 默认十进制
    - 二进制以0b或0B开头
        - 如0b010, -0B101
    - 八进制以0o或0O开头
        - 如0o123, -0O456
    - 十六进制以0x或0X开头
        - 如0x9a, -0X89
- 浮点类型
    - Python语言用一个53位二进制表示小数部分，取值范围数量级约-10^308至10^308，精度数量级为10^-16
    - 浮点数间运算存在不确定尾数
        - 如>>>0.1 + 0.2 == 0.3返回False
    - 要使用函数**round(x, d)**，对x四舍五入，d是小数截取位数
        - 如>>>round(0.1 + 0.2, 1) == 0.3返回True
- 科学计数法表示: 使用e或E作为幂的符号，以10为基数，格式：\<a>e\<b>表示a*10^b
    - 如4.3e-3 = 0.0043, 9.6E5 = 960000.0
- 复数表示：
    - 如复数z = 1.23e-4 + 5.6e + 89j，可以用**z.real**获得实部，用**z.imag**获得虚部，用**z.conjugate()**返回共轭复数
- 数值运算操作符
    - **+ - * /** 和数学运算中一样
        - 如10 / 3 = 3.3333333333
    - **//** 两个斜杠表示整数除
        - 如10 // 3 = 3
    - **+ -** 正负号和数学中定义一样
    - **%** 取模运算
    - **\*\*** 幂运算，**x ** y** 就是x的y次幂，和pow(x, y)的功能一样
        - 如5 ** 2 = 25, 25 ** 0.5 = 5
    - 在运算符后面加个=可以使代码比较简洁
        - 如x += y等价于x = x + y, x /= y等价于x = x / y, x //= y等价于x = x // y, x **= y等价于x = x ** y
    - 混合数字类型计算一般会生成一种最宽的类型
        - 如123 + 4.0 = 127.0
    - 关系运算符 可以连用，一般用于同类型对象之间值的大小比较，或者测试集合间的包含关系
        - 如1 < 3 < 5返回True
        - 如'Hello' > 'world'返回False
        - 如[1, 2, 3] < [1, 2, 4]返回True
        - 如'hello' > 3出错，不同类型不能比较
        - 如{1, 2, 3} < {1, 2, 3, 4}返回True，即是子集
    - **is** 同一性测试运算符，用来测试两个对象是否是同一个，如果是，二者具有相同的内存地址
        - 如3 is 3返回True
        - 如x = [300, 300, 300] x[0] is x[1]返回True
        - 如x = [1, 2, 3]   y = [1, 2, 3]   x is y返回False，因为上面形式创建的x和y不是同一个列表对象
    - 位运算符 只能用于整数
        - **<<** 左移位运算符，右侧补0，每左移一位相当于乘以2
        - **>>** 右移位运算符，左侧补0，没右移一位相当于整除以2
        - **&** 位与运算符
        - **|** 位或运算
        - **^** 位异或运算

## 函数
- **print("输出内容")** 在屏幕输出内容
    - 增加end=""参数表示输出后不换行，否则默认换行
        - 如print("hello world!",end="")
    - 将输出内容用逗号隔开，在输出结果中，内容之间会被空格隔开
        - 如print("hello world", 1)，输出结果为hello world 1
- **turtle** 海龟绘图库函数
    - **import turtle** 在代码开头引入turtle库
    - 两种引入方法
        - **from turtle import\*** 后面可以不用加turtle.，直接使用库中函数。缺点是可能函数重名
            - 如fd(40)
        - **import turtle as tt** *(最优)* 将turtle引入并起个别名tt，后面可以通过别名使用函数
            - 如tt.fd(40)
    - **turtle.setup(width, height, startx, starty)** 设置绘图窗体的启动长宽(width, height)和位置(startx, starty)(后两个参数可以省略，则默认窗体位置在屏幕正中央)
        - 如turtle.setup(650, 350, 200, 200)
    - **turtle.goto(x, y)** 将海龟移动到绝对坐标(x, y)并留下移动轨迹
        - 绝对坐标：原点(0, 0)位于窗体的正中央，单位为像素点，x轴方向水平向右，y轴方向竖直向上
    - **turtle.seth(angle)** 设定海龟面对的方向为绝对方向angle
        - 又名**turtle.setheading(angle)**
        - 如turtle.seth(-40)或turtle.setheading(-40)
    - **turtle.fd(d)** 使海龟向前运动d个像素
        - 又名**turtle.forward(d)**
        - 海龟坐标：在海龟视角下的方向
        - 如果d为负数向后运动
        - 如turtle.fd(40)或turtle.forward(40)
    - **turtle.bk(d)** 使海龟向后运动d个像素
        - 如turtle.bk(40)
    - **turtle.circle(r, extent)** 以海龟左侧距离r的点为圆心，移动角度extent画出一段圆弧
        - 如果r值为负数，则圆心在turtle的右边
        - 如果省略参数extent，默认画一个圆
    - **turtle.left(angle)** / **turtle.right(angle)** 使海龟向左/右转角度angle
    - **turtle.pencolor(color)** 设置画笔颜色，参数color为颜色名字符串(小写)或RGB值
        - RGB色彩体系：由红绿蓝构成的色彩，默认采用小数值表示(3个0\~1的小数)，可以设置为整数值表示(3个0\~255的整数)。常用RGB色彩：
            - 白色 white：(255, 255, 255)或(1, 1, 1)
            - 黄色 yellow：(255, 255, 0)或(1, 1, 0)
            - 洋红 magenta：(255, 0, 255)或(1, 0, 1)
            - 青色 cyan：(0,255, 255)或(0, 1, 1)
            - 蓝色 blue：(0, 0, 255)或(0, 0, 1)
            - 黑色 black：(0, 0, 0)
            - 海贝色 seashaell：(255, 245, 238)或(1, 0.96, 0.93)
            - 金色 gold：(255, 215, 0)或(1, 0.84, 0)
            - 粉红色 pink：(255, 192, 203)或(1, 0.75, 1.80)
            - 棕色 brown：(165, 42, 42)或(0.65, 0.16, 0.16)
            - 紫色 purple：(160, 32, 240)或(0.63, 0.13, 0.94)
            - 番茄色 tomato：(255, 99, 71)或(1, 0.39, 0.28)
        - 如turtle.pencolor(0.63, 0.13, 0.94)或turtle.pencolor((0.63, 0.13, 0.94))(后者将purple的RGB值形成一个元组值，放进参数color里)
    - **turtle.colormode(mode)** 设置RGB颜色表示模式
        - 设置mode = 1，用小数0\~1表示RGB色彩；设置mode = 255，用整数0~255表示RGB色彩
    - **turtle.penup()** 抬笔，使海龟飞行，不留下痕迹
        - 又名**turtle.pu()**
    - **turtle.pendown()** 落笔，使海龟在移动时留下痕迹
        - 又名**turtle.pd()**
    - **turtle.pensize(width)** 设置画笔宽度width，即海龟的腰围
        - 又名**turtle.width(width)**
    - **turtle.done()** 设置绘制完成后不自动关闭绘图窗体推出
- **range(M, N)** 产生M到N - 1的循环计数序列
    - 如range(0, 4)产生0, 1, 2, 3共4个整数
    - **range(N)** 忽略第一个参数则产生0到N - 1的循环计数序列
        - 如range(3)产生0, 1, 2共3个整数
    - **for i in range(M, N) 后面缩进加语句** 对于每个在M和N - 1中的数i，循环执行一次后面的语句
- **input("提示语")** 从控制台获取信息并返回
    - 如input("please input a number")
- 列表操作函数
    - **list(obj)** 将元组、range对象、字符串或其它类型的可迭代对象类型数据obj转换为列表
        - 如list(range(1, 10, 2))生成[1, 3, 5, 7, 9]; list('hello world')生成['h', 'e', 'l', 'l', 'o', ' ', 'w', 'o', 'r', 'l', 'd']; x = list()创建空列表x
    - 列表常用方法
        - Python中任何一个变量，包括列表，都是一个对象，是一种\<a>，存在着一些方法。方法特指\<a>.\<b>()风格中的函数\<b>()，本身也是函数，必须采用\<a>.\<b>()的形式使用
        - **list.append(x)** 将元素x添加到列表list尾部
        - **list.extend(L)** 将列表L中所有元素添加至列表list尾部
        - **list.insert(index, x)** 在列表list指定位置index处添加元素，原本在该位置及后边的所有元素后移一个位置
        - **list.remove(x)** 在列表list中删除首次出现的指定元素x，该元素之后的所有元素前移一个位置
        - **list.pop(index)** 删除并返回列表list中下标为index(默认为-1)的元素
        - **list.clear()** 删除列表list中的所有元素，但保留列表对象
        - **list.index(x)** 返回列表list中第一个值为x的元素的下标，若不存在值为x的元素则抛出异常
        - **list.count(x)** 返回指定元素x在列表list中出现的次数
        - **list.reverse()** 对列表list中所有元素进行逆序
            - 此方法是将元素原地逆序，而使用内置函数reversed()则是对列表元素进行逆序排列并返回迭代对象
        - **list.sort(key=None, reverse=False)** 对列表list中的元素进行排序，key用来指定排序依据，reverse决定升序(False)还是降序(True)
            - 此方法是将元素原地排序，而使用内置函数sorted()则是对列表元素进行排序并返回新列表
            - 如aList.sort(key=lambda x:len(str(x)))是按转换成字符串的长度排序
        - **list.copy()** 返回列表list的浅复制
- 字符串操作函数
    - **len(x)** 返回字符串x的长度
        - 如len("一二三456") = 6
    - **eval(x)** 评估函数，去掉字符串x最外侧引号并以python的语法执行余下语句
        - 如eval("1")返回1; eval("1+2")返回3; eval('"1+2"')返回字符串"1+2"(因为只去掉最外层引号)
    - **str(x)** 返回任意类型x对应的字符串形式(即在两端加双引号)(与函数eval(x)作用相反)
        - 如str(1.23) = "1.23"; str([1, 2]) = "[1, 2]"
    - **repr(obj)** 返回对象obj的规范化字符串表示形式，对于大多数对象有eval(repr(obj))==obj
    - **hex(x)** / **oct(x)** 返回整数x的十六进制/八进制小写形式字符串
        - 如hex(425) = "0x1a9"; oct(425) = "0o651"
    - **chr(u)**（u为Unicode编码）返回其对应的字符
    - **ord(x)**（x为字符）返回其对应的Unicode编码
    - 字符串处理方法
        - **str.lower(x)** / **str.upper(x)** 返回字符串x的副本，且全部字符小/大写
            - 如"AbCdEfGh".lower() 返回 "abcdefg"
        - **str.split(sep)** 返回一个列表，由str根据sep被分隔的部分组成
            - 如"a,b,c".split(",") 返回 ['A', 'B', 'C']
        - **str.count(sub)** 返回子串sub在str中出现的次数
            - 如"an apple a day".count("a") 返回 4
        - **str.replace(oldPart, newPart)** 返回字符串str副本，所有oldPart子串被替换为newPart
            - 如"python".replace("n", "n123.io") 返回 "python123.io"
        - **str.center(width[,fillchar])** 返回字符串str根据宽度width居中，fillchar可选
            - 如"python".center(20, "=") 返回 '=======python======='
        - **str.strip(chars)** 从str中去掉chars中列出的所有字符
            - 如"= python=".strip(" =up") 返回 "ytho"
        - **str.join(iter)** 在iter变量除最后元素外每个元素后增加一个str，主要用于字符串分隔
            - 如",".join("12345") 返回 "1,2,3,4,5"
        - **str.format()** 使字符串str格式化，即进行格式表达
            - str中的{}名为槽，是占位信息符，按顺序填入format()括号内的参数
                - 如"{}:计算机{}的CPU占用率为{}%".format("2018-10-10", "C", 10)返回 "2018-10-10:计算机{}的CPU占用率为10%"
            - 槽中还可以指定添加参数的位置
                - 如"{1}:计算机{0}的CPU占用率为{2}%".format("2018-10-10", "C", 10)返回 "C:计算机2018-10-10的CPU占用率为10%"
            - 槽内部对格式化的配置方式：{<参数位置>:<填充符号><对齐方式><宽度><,><.精度><类型>}
                - 其中<填充符号>中填用于填充的单个字符，<对齐方式>中填<为左对齐，>为右对齐，^为居中对齐
                    - 如"{0:=^20}".format("python")中，等号是填充符号，^是居中对齐，20是宽度
                - 类型中：b二进制，c字符(Unicode字符集)，d十进制，o八进制，x小写十六进制，X大写十六进制; e小写科学计数法，E大写科学计数法，f浮点数，%百分数
- **abs(x)** 取x的绝对值
    - 如abs(-10.01) = 10.01
- **divmod(x, y)** 返回(x // y, x % y)，商余函数，同时输出商和余数
    - 如divmod(10, 3) = (3, 1)
- **pow(x, y[, z])** 返回(x ** y) % z，幂余函数，[...]表示参数z可省略
    - 如pow(3, pow(3, 99), 10000) = (3 ** (3 ** 99)) % 10000 = 4587，表示取3的3的99次方的最后四位
- **round(x[, d])** 对x四舍五入，d是小数截取位数，默认值为0，常用于浮点数间运算
    - 如>>>round(0.1 + 0.2, 1) == 0.3返回True
- **max(x1, x2, x3, ..., xn)**, **min(x1, x2, x3, ..., xn)** 取最大/最小值，返回x1, x2, x3, ..., xn中的最大/最小值，n不限
    - 如max(1, 9, 5, 4, 3) = 9，min(1, 9, 5, 4, 3) = 1
- **sum(x, start=0)** 返回序列x中所有元素之和，返回start+sum(X)
    - 如sum(range(1, 11), 5)，返回5+sum(range(1, 11))
    - 如sum([[1, 2], [3], [4]], [])，返回[1, 2, 3, 4]，但此操作占用空间较大，慎用
- **int(x)** 将x变成整数，舍弃小数部分（不进行四舍五入）
    - 如int(123.45) = 123，int("123") = 123
- **float(x)** 将x变成浮点数，增加小数部分
    - 如float(12) = 12.0，float("1.23") = 1.23
- **complex(x)** 将x变成复数，增加虚数部分
    - 如complex(4) = 4 + 0j
- **all(iterable)** 如果对于可迭代对象对象中所有元素x都等价于True，或者可迭代对象为空，则返回True
- **any(iterable)** 只要可迭代对象iterable中存在元素使得x等价于true，返回True，可迭代对象为空，返回False
- **ascii(obj)** 把对象obj转换为ASCII码表示形式，必要的时候使用转义字符来表示特定的字符
- **bin(x)** 把整数x转换为二进制串表示形式

- **callable(obj)** 测试对象obj是否可调用。类和函数是可调用的，包含__call__()的方法的类的对象也是可调用的
- **compile()** 用于把python代码编译成可被exec()或eval()函数执行的代码对象
- **delattr(obj, name)** 删除属性，等价于del obj.name
- **dir(obj)** 返回指定对象或模块的成员列表，如果不带参数则返回当前作用域内所有标识符
- **enumerate(iterable)** 返回包含元素形式为(0, iterable[0]), (1, iterable[1]), (2, iterable[2]),...的迭代器对象
    - 如note = [1, 2, 3] solmization = ['Do', 'Re', 'Mi'] enumerable(note, solmization) = [(1, 'Do'), (2, 'Re'), (3, 'Mi')]
    - 如for item in enumerate('abc'): print(item)，输出结果为(0, 'a')(1, 'b')(2, 'c')

- **quit()** 退出当前解释器环境

- **frozenset([x])** 创建不可变的集合对象

- **list([x])/set([x])/tuple([x])/dict([x])** 把对象x转换为列表、集合、元组或字典返回，或生成空列表、空集合、空元组、空字典

    - 如aList = [1, 2, 3], bList = [4, 5, 6], cList = zip(aList, bList)，得到cList为\<zip object at \<所在地址\>\>，list(cLst)得到[(1, 4), (2, 5), (3, 6)]
